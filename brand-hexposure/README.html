What's in this package

The Hexposure Brand board (all directions, Iris = 02) and the Iris Logo Options canvas, with their React/JSX sources. Open the .html files; both have a dark/light toggle.

The marketing site with friendly filenames — Hexposure, Platform, Why Hexposure, Privacy Policy, Terms — plus site.css / site.js. Theme toggle built into the nav.

Same site, deploy-ready (lowercase URLs, .nojekyll) for GitHub Pages / static hosting.

Standalone SVGs: mark (gradient / on-dark / on-light), favicon tile, and wordmark lockups for both contexts.
tokens.css
All colour tokens for both themes as CSS custom properties, ready to drop into any project.
